# Dashboard de Reporte de ventas
Dashboard de informe de ventas realizadas del TioTech realizado en Power BI para mostrar los datos más relevantes de la base de datos, que ayuden a la toma de decisiones empresariales.

Procedimiento de la realización paso a paso:

1. Carga del archivo Excel a la vista de trabajo. Al trabajar con una sola tabla, en este caso no fue necesario realizar el modelado de datos entre tablas.
  
2. Preparación y transformación de los datos con la herramienta Power Query. Dentro de está etapa se realizaron las siguientes modificaciones:
- Cambio de tipo de variable de algunos campos.
- Reemplazo del nombre de ciertas variables por otras

3. Incrustación de diferentes visualizaciones para poder visualizar mejor el contenido de la tablas. Entre estas incluimos:
- Una tabla de ventas totales por país, donde se coloca a cada país junto con el importe total correspondiente.
- Un gráfico de dispersión que señala la correlación entre las unidades vendidas por cada mes.
- Un gráfico circular que divide la venta total entre los años 2020, 2021 y 2022.
- Un gráfico de columnas apiladas que señala las ventas totales realizadas por mes, cada mes se divide en los años 2020, 2021 y 2022, indicando que año tuvo mayores ventas en un mes dado.
- Un gráfico de barras apiladas que señala las ventas totales realizadas por país, dividiendo cada país entre los años 2020 al 2022, indicando el porcentaje de ventas que tuvo cada año para cada país.

4. Colocación de tarjetas que indican totales claves, como el total de unidades, total de costo general, total de venta general. Y de filtros de segmentación, para: canal de venta, año y país.
